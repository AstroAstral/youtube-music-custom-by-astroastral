# YouTube Music Custom by AstroAstral

**Status: Alpha**

Este é um UserStyle para a extensão **Stylus** que customiza completamente a interface do YouTube Music
(com fundo dinâmico, blur/vidro, player customizado, barra de progresso avançada e ajustes visuais).

**Aviso**
- Este projeto está em fase **alpha**
- O código ainda está sendo refatorado
- Algumas regras podem mudar ou quebrar após atualizações do YouTube Music

## Requisitos
- Navegador baseado em Chromium ou Firefox
- Extensão Stylus

## Instalação
1. Instale a extensão Stylus
2. Crie um novo estilo
3. Cole o conteúdo do arquivo `style.css`
4. Ative para `music.youtube.com`
